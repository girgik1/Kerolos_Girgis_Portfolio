// Embedded Systems · Arduino · C
PROXIMITY-ACTIVATED FAN
Expand Project ▼
▶ Live Demo — Proximity Fan Activation

Overview

Designed and programmed a proximity-activated cooling fan using an Arduino microcontroller and HC-SR04 ultrasonic distance sensor. When an object is detected within 15 cm, the fan activates automatically via GPIO — demonstrating real-time sensor polling, embedded control logic, and hardware-software integration.

How It Works

  • Arduino sends a 10 µs trigger pulse on pin 8 to the HC-SR04
  • Echo return time measured via pulseIn() with 30 ms timeout
  • Distance calculated: time × 0.0343 / 2 (cm)
  • Fan on pin 10 toggled HIGH/LOW based on distance threshold
  • Serial monitor outputs live distance data at 9600 baud
  • 100 ms polling loop for stable, responsive actuation

ProximityFan.ino — Full Source 
const int TriggerOutput = 8;
const int ScanInput = 7;   // Echo pin
const int Fan = 10;

unsigned long TimeFromTarget = 0;
float distance = 0.0;

void setup() {
  pinMode(ScanInput, INPUT);
  pinMode(TriggerOutput, OUTPUT);
  pinMode(Fan, OUTPUT);

  digitalWrite(TriggerOutput, LOW);
  digitalWrite(Fan, LOW);

  Serial.begin(9600);
}

void loop() {
  // Make sure trigger starts LOW
  digitalWrite(TriggerOutput, LOW);
  delayMicroseconds(2);

  // Send 10 us trigger pulse
  digitalWrite(TriggerOutput, HIGH);
  delayMicroseconds(10);
  digitalWrite(TriggerOutput, LOW);

  // Wait for echo pulse, timeout after 30 ms
  TimeFromTarget = pulseIn(ScanInput, HIGH, 30000);

  if (TimeFromTarget == 0) {
    Serial.println("No echo received");
    distance = 0;
  } else {
    distance = TimeFromTarget * 0.0343 / 2.0;
    Serial.print("Distance from target: ");
    Serial.println(distance);
  }

  if (distance > 0 && distance <= 15) {
    digitalWrite(Fan, HIGH);
  } else {
    digitalWrite(Fan, LOW);
  }

  delay(100);
}
